# Hero Foto Upload Instructies

## Marktkraam Afbeelding als Hero Foto

Volg deze stappen om de marktkraam afbeelding als hero foto op de homepage te gebruiken:

### Stap 1: Upload de Afbeelding

1. Ga naar **Shopify Admin** → **Content** → **Files**
2. Klik op **Upload files**
3. Upload de marktkraam afbeelding
4. Noteer de bestandsnaam of URL

### Stap 2: Pas de Homepage aan

1. Ga naar **Online Store** → **Themes**
2. Klik op **Customize** bij je actieve theme
3. Scroll naar de **Image banner** sectie (bovenaan de homepage)
4. Klik op de **Image banner** sectie om deze te bewerken

### Stap 3: Configureer de Hero Sectie

**Image Settings:**
- **Image**: Selecteer de geüploade marktkraam afbeelding
- **Image overlay opacity**: 30-40% (voor betere tekst leesbaarheid)
- **Image height**: **Large** (voor maximale impact)
- **Image behavior**: **None** of **Zoom-in** (voor subtiele animatie)

**Content Settings:**
- **Desktop content position**: **Middle-center** (centraal)
- **Desktop content alignment**: **Center**
- **Show text box**: **Aan** (voor tekst overlay)
- **Color scheme**: Kies een schema dat goed contrasteert

**Mobile Settings:**
- **Mobile content alignment**: **Center**
- **Show text below**: **Uit** (behoud overlay op mobiel)

### Stap 4: Pas de Content aan

**Heading Block:**
- Tekst: "De Notenmarkt" of "Welkom bij De Notenmarkt"
- Heading size: **H0** (extra groot)

**Text Block:**
- Tekst: "Vers gebrand, zorgvuldig geselecteerd" of je eigen tagline
- Text style: **Subtitle**

**Buttons Block:**
- Button 1 label: "Ontdek Onze Noten"
- Button 1 link: `/collections/all` of `/products`
- Button 1 style: **Primary** (accent kleur)
- Button 2 label: "Ons Verhaal" (optioneel)
- Button 2 link: `/pages/about` (als je een about pagina hebt)
- Button 2 style: **Secondary**

### Stap 5: Opslaan

1. Klik op **Save** (rechtsboven)
2. Preview je homepage
3. Test op mobiel en desktop

## Tips

- **Afbeelding Formaat**: Gebruik een hoge resolutie afbeelding (minimaal 1920x1080px)
- **Bestandsgrootte**: Optimaliseer de afbeelding voor web (max 500KB voor snelle laadtijden)
- **Aspect Ratio**: 16:9 of 21:9 werkt goed voor hero secties
- **Tekst Contrast**: Zorg dat de tekst goed leesbaar is over de afbeelding (gebruik overlay opacity)

## Custom Styling

De custom styling in `denotenmarkt-custom.css` zorgt automatisch voor:
- Warme kleuren en fonts
- Rounded buttons
- Mooie shadows en transitions

Je hoeft geen extra CSS toe te voegen - alles werkt automatisch met de theme editor instellingen!
